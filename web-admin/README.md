# Panel de Administración - E-commerce Descentralizado

Panel de administración para el sistema de e-commerce descentralizado basado en blockchain.

## Características

- Gestión de empresas registradas en la blockchain
- Administración de productos por empresa
- Gestión de clientes
- Integración con MetaMask
- Interfaz responsive con Tailwind CSS
- Conexión con contratos inteligentes de Ethereum

## Tecnologías

- Next.js 16 con App Router
- React 19
- TypeScript
- Tailwind CSS
- Ethers.js v6
- Contratos inteligentes Solidity

## Requisitos

- Node.js 18+
- npm o yarn
- MetaMask (para interacción con la blockchain)
- Anvil o Hardhat (para red local de desarrollo)
